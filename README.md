用于 MT 管理器语法文件 (.mtsx) 的语言服务器(LS), 使用语言服务器协议 (LSP)

# 支持特性
- [x] 补全字段
- [x] 补全内建匹配器、格式化器、收缩器
- [x] 补全内建模式助手函数
- [x] 补全内建颜色
- [x] 补全用户颜色
- [x] 补全引入匹配器
- [x] 补全引入正则
- [x] 检查重复字段
- [ ] 检查未知字段
- [ ] 查看字段文档
- [ ] 查看用户定义的匹配器、正则、颜色与文档
- [ ] 跳转用户定义的匹配器、正则、颜色的定义和引用

# 编辑器
- vim: 使用 [mtsyntax.vim](https://github.com/A4-Tacks/mtsyntax.vim) 获取语法高亮
